readNWMdata {nwmHistoric}R Documentation

NWM Reanalysis Extraction

Description

Download hourly flow values for an NHD COMID from the National Water Model version 2.0. Returned data is availiable between "1993-01-01" and "2017-12-31" but can be subset using a startDate and endDate.

Usage

readNWMdata(
  comid = NULL,
  siteNumber = NULL,
  startDate = "1993-01-01",
  endDate = "2018-12-31",
  tz = "UTC",
  version = 2
)

Arguments

comid

a NHD common identifier

siteNumber

a USGS NWIS site number (eight digits)

startDate

a start date (YYYY-MM-DD)

endDate

an end date (YYYY-MM-DD)

tz

the desired timezone of the data. Can be found with 'OlsonNames()'

version

the NWM version to extract (current = 1.2 or 2.0 (default))

Value

data.frame

[Package nwmHistoric version 0.0.0.9000 Index]
