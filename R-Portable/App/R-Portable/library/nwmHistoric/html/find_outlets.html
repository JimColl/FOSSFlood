find_outlets {nwmHistoric}R Documentation

Identify NHD outlets from flowlines

Description

Convert flowline NHD realization to outlets.

Usage

find_outlets(flowlines)

Arguments

flowlines

a sf LINESTRING or MULTILINESTRING object

Value

the input flowlines object with geometry changed to POINT objects representing the oulets
