discover_nwis {nwmHistoric}R Documentation

Discover USGS NWIS Stream Gages

Description

discover_nwis returns a POINT feature class of NWIS gages reporting parameter code '00060' (streamflow) for an Area of Interest that ALSO report NWM data. Data is accessed through the NWIS web portal.

Usage

discover_nwis(AOI = NULL)

Arguments

AOI

A spatial geometry

Value

sf object
