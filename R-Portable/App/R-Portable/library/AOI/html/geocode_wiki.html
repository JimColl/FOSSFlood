geocode_wiki {AOI}R Documentation

Geocoding Events

Description

A wrapper around the Wikipedia API to return geo-coordinates of requested inputs.

Usage

geocode_wiki(event = NULL, pt = FALSE)

Arguments

event

character. a term to search for on wikipeida

pt

logical. If TRUE point geometery is appended to the returned list()

Value

aa data.frame of lat/lon coordinates

Author(s)

Mike Johnson

Examples

## Not run: 
   ## geocode an Agency
     geocode_wiki("NOAA")

 ## geocode an event
     geocode_wiki("I have a dream speech")

 ## geocode a n event
     geocode_wiki("D day")

 ## geocode a product
     geocode_wiki("New York Times")

 ## geocode an event
     geocode_wiki("Hurricane Harvey")


## End(Not run)
