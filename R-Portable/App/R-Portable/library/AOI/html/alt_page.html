alt_page {AOI}R Documentation

Alternate Page Finder

Description

Find linked pages to a wikipedia call

Usage

alt_page(loc, pt = FALSE)

Arguments

loc

a wikipedia structured call

pts

logical. If TRUE point geometery is appended to the returned list()

Value

at minimum a data.frame of lat, long

Author(s)

Mike Johnson

Examples

## Not run: 
alt_page("Twin_towers")

## End(Not run)
