map_flood {FloodMapping}R Documentation

Map flood extents from processed data

Description

This function takes a folder of processed data and generates a HAND inudation raster for each timestep

Usage

map_flood(hand.path, catchment.path, rating.path, flows.path, add = 0)

Arguments

hand.path

a path to HAND tif

catchment.path

a path to catchmask tif

rating.path

a path to rating fst file

flows.path

a path to a flows nc file

add

numeric. A uniform depth to add to simulated inudation

Value

a raster stack of innudation maps
