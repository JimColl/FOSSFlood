download_ut {FloodMapping}R Documentation

Download HAND, RC, and Cathmask rasters by HUC6

Description

This function downloads staged HAND data to a local directory.

Usage

download_ut(HUC6, outdir, type = "all")

Arguments

HUC6

A HUC6 unit to download

outdir

The folder path where data should be downloaded and extracted

type

The type of data to download. Options include 'catchmask'. 'hand', or 'rating curve'. If 'all' is used (DEFAULT) all three type are downloaded.
