snotel {HydroData}R Documentation

Snotel Stations

Description

snotelNRCS SNOTEL station metadata

Usage

snotel

Format

a dataframe instance, 1 row per station with columns:

  • 'NETWORK': A integer Network of interest

  • 'STATE': A integer State abbriviation

  • 'NAME': A character Site name

  • 'START.DATE': A integer Day of first measurement

  • 'LAT': A integer Station Latitude

  • 'LONG': A character Station Longitude

  • 'ELEV': A character Station Elevation

  • 'COUNTY': A character County

  • 'HUC12.NAME': A character HUC 12 Name

  • 'HUC12.ID': A character HUC 12 code
